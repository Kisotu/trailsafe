## 📍 Overview
 - Trailsafe is a hikers safety companion app that allows a user specifically hikers to intract with geodata information on a map. They can add, view and edit pins on marker based on their experience on slected trails.

---

## 👾 Features

 - User registration
 - Usr login
 - View pins on maps
 - Add pins to map
 - Add details to markers

## 🚀 Getting Started

### ☑️ Prerequisites

Before getting started with trailsafe, ensure your runtime environment meets the following requirements:

- **Programming Language:** JavaScript
- **Package Manager:** Npm


### ⚙️ Installation

Install trailsafe using one of the following methods:

**Build from source:**

1. Clone the trailsafe repository:
```sh
❯ git clone https://github.com/kisotu/trailsafe
```

2. Navigate to the project directory:
```sh
❯ cd trailsafe
❯ cd client
❯ cd server
```

3. Install the project dependencies:


**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm install
```




### 🤖 Usage
Run trailsafe using the following command:
**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm start
```


### 🧪 Testing
Run the test suite using the following command:
**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh

```
